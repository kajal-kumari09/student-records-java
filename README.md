# 🎓 Student Management System (Java)

This is a simple Java-based Student Management System project that performs basic CRUD operations.

---

## 🚀 Features
- Add Student
- View Students
- Update Student
- Delete Student

---

## 🛠️ Technologies Used
- Java
- ArrayList
- Scanner Class

---

## 📂 Project Structure
Student.java → Student model class  
StudentManagement.java → Main logic (CRUD operations)  
StudentApp.java → Main entry point

---

## ▶️ How to Run
1. Compile all Java files:
   javac *.java

2. Run the program:
   java StudentApp

---

## 📌 Purpose
This project is created for learning Java basics and understanding object-oriented programming (OOP) concepts.
